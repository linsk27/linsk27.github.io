AI-v0.dev- 是一个用 Next.js、React、TypeScript、Tailwind CSS 和 shadcn/ui 搭出来的 AI 学习仪表盘原型。它把 3 个月 AI 工程师冲刺计划拆成时间线、项目卡片、知识体系、里程碑、任务进度和本地数据管理。

适合谁

  • 想看 v0.dev 生成页面如何继续整理成真实项目的人。
  • 正在做 AI learning dashboard、Next.js dashboard 或学习计划工具的人。
  • 想参考 shadcn/ui、Tailwind 和 Radix UI 组合方式的前端开发者。
  • 需要 AI UI prototype、项目卡片、进度追踪和本地状态管理界面的人。

项目价值

这个项目的重点是产品原型表达。它不是默认模板仓库，而是把学习计划、任务、项目、资源和里程碑组合成一个可交互的仪表盘。对于搜索 AI UI prototype、v0.dev workflow、Next.js dashboard、Tailwind interface design 或 AI learning dashboard 的人，这个页面能更快说明项目价值。

技术方向

  • Next.js
  • React
  • TypeScript
  • Tailwind CSS
  • shadcn/ui
  • Radix UI
  • localStorage 状态管理
  • v0.dev workflow
