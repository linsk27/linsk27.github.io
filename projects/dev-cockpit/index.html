Dev Cockpit 本地开发工作台

Dev Cockpit 是 linsk27 的本地开发工作台项目，用来集中查看项目状态、启动命令、日志、端口、Git 状态和 AI coding 上下文。

Dev Cockpit 是我用来整理本地开发现场的工具。它关注的问题很直接：当一个人同时维护多个项目时，如何快速知道项目怎么启动、端口是否占用、命令是否可用、Git 状态是否干净，以及哪些上下文需要交给 AI coding agent。

适合谁

  • 经常在多个前端、全栈或 AI 项目之间切换的个人开发者。
  • 需要快速恢复项目现场、少翻 README、少找命令的人。
  • 想把项目状态、日志、端口和 Git 信息集中到一个界面里的人。
  • 正在尝试 AI coding workflow，希望把项目上下文整理得更稳定的人。

项目价值

Dev Cockpit 不是一个简单的启动器。它更像一个开发现场看板：把项目入口、常用命令、运行日志、端口状态、Git 状态和 AI 上下文放到同一个位置，让开发者少做重复判断。

如果你搜索的是 local development dashboard、developer workflow tools、AI coding context manager、Electron Vue dashboard 或 project state manager，这个项目会比普通练手页面更接近真实需求。

技术方向

  • TypeScript
  • Vue
  • Electron
  • 本地项目扫描
  • 命令与日志管理
  • Git 状态读取
  • AI coding 上下文整理
