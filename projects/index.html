Dev Cockpit

Dev Cockpit 是一个本地开发工作台，用来集中查看项目状态、启动命令、日志、端口、Git 状态和 AI 上下文。它适合经常在多个项目之间切换、需要快速恢复开发现场的个人开发者。

如果你正在找 local development dashboard、developer workflow tools、AI coding context manager 或 Electron + Vue 的项目，可以先看 Dev Cockpit 详情页，再进入 GitHub 仓库。

ProofPR

ProofPR 是一个 GitHub Action 和 CLI，用来整理 Pull Request 中的证据信号，包括 secrets、依赖风险、CI 权限、MCP 安全和 review 线索。目标是让开源维护者更快判断一个 PR 是否值得合并、需要补充什么证据。

如果你关注 pull request review automation、GitHub Actions security、PR triage、AI generated code review 或 maintainer tools，可以先看 ProofPR 详情页，再进入 GitHub 仓库。

skills-manage

skills-manage 是一个 Tauri、React 和 TypeScript 桌面应用，用来集中管理 AI coding agent skills，覆盖中央技能库、平台安装、项目扫描、技能市场、GitHub 导入和技能集合。

如果你正在找 AI coding agent skills manager、Claude Code skills、Codex CLI skills、Cursor skills、Agent Skills desktop app 或 Tauri React desktop app，可以先看 skills-manage 详情页，再进入 GitHub 仓库。

AI-Vue3-python-flask-Blog

AI-Vue3-python-flask-Blog 是一个 Vue 3、Flask、MySQL 和 AI 能力组合的博客系统，覆盖前端页面、后端接口、数据持久化和内容管理。它适合作为课程设计、毕业设计、全栈项目练习和二次开发基础。

如果你在搜索 Vue Flask full stack project、Vue3 Flask MySQL blog system、AI blog system、graduation project 或 course design，可以先看 AI Vue Flask Blog 详情页，再进入 GitHub 仓库。

AI-v0.dev-

AI-v0.dev- 是一组 TypeScript 前端原型，用来探索 AI 产品界面、快速页面生成、组件组合和 v0.dev 风格的交互表达。它更适合看界面实验、AI UI prototype、Next.js、Tailwind 和前端产品想法。

如果你想看 AI learning dashboard、Next.js dashboard、v0.dev workflow、shadcn/ui 或 Tailwind 产品原型，可以先看 AI-v0.dev- 详情页，再进入 GitHub 仓库。
