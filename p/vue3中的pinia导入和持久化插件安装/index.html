vue3中的pinia导入和持久化插件安装

Vue Pinia

2024-10-12

一. yarn的安装

  1. 使用Windows powershell（管理员）A命令行： npm install -g yarn，并且使用yarn -v查看是否安装成功
  2. 打开控制面板的系统和安全，打开系统，进入高级系统设置，找到yarn的bin安装目录位置，进行环境变量的植入配置
  3. 即可使用yarn命令进行安装

二. 使用yarn安装pinia，并且导入vue3项目

  1. 打开vue3的项目终端，输入yarn add pinia完成安装
  2. 在main.js导入pinia包import { createPinia } from 'pinia'，进行生成pinia配置const pinia = createPinia()
  3. 即可使用pinia的状态仓库

三. 安装持久化插件，并且导入

  1. 打开vue3 的项目终端，输入yarn add pinia-plugin-persistedstate完成安装
  2. 在main.js导入该持久化插件包import piniaPluginPersistedstate from 'pinia-plugin-persistedstate' 在pinia使用该插件输入pinia.use(piniaPluginPersistedstate)
  3. 即可实现本地数据持久化
